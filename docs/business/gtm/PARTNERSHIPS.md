## État actuel

*Quickfund a-t-il des partenariats ?*

**Situation :** Aucun partenariat formel actuellement.

| Élément | Valeur |
|---------|--------|
| Partenaires actifs | 0 |
| CA via partenaires | 0€ |
| Responsable partenariats | Fondateur (non prioritaire) |
| Programme formalisé | ❌ Non |

*Pourquoi pas de partenariats ?* L'entreprise a fonctionné en "pilote automatique" pendant 2 ans. La priorité était de valider le PMF, pas de développer des partenariats. C'est un axe de croissance pour plus tard.

## Rachat de crédits comme acquisition

*Le rachat de crédits est-il un partenariat déguisé ?*

**Stratégie mentionnée dans COMPETITIVE.md :** Utiliser le rachat de crédits comme outil d'acquisition.

| Élément | Description |
|---------|-------------|
| Concept | Racheter les crédits des concurrents |
| Avantage | Acquérir des clients avec historique de remboursement |
| Coût | Potentiellement à perte court terme |
| ROI | Fidélisation long terme |

*Ce n'est pas un partenariat formel mais une stratégie d'acquisition agressive.*

## Pourquoi pas de partenariats formels ?

*Quels sont les freins au développement de partenariats ?*

| Frein | Impact |
|-------|--------|
| Équipe de 1 personne | Pas de temps dédié |
| Priorité structuration | Les métriques d'abord |
| Pas de programme formalisé | Difficile d'onboarder des partenaires |
| Notoriété faible | Difficile d'attirer des partenaires |

## Critères pour un bon partenariat

*Si un partenariat se présentait, quels critères ?*

| Critère | Importance |
|---------|:----------:|
| Apporte des clients qualifiés | Haute |
| Coût d'acquisition raisonnable | Haute |
| Aligné avec les valeurs (transparence) | Moyenne |
| Ne nécessite pas trop de ressources | Haute |
| ROI mesurable | Haute |

### Deal-breakers

| Critère | Raison |
|---------|--------|
| Commission trop élevée | Rogne les marges |
| Exclusivité contraignante | Limite la croissance |
| Besoins techniques lourds | Pas les ressources |
| Clients de mauvaise qualité | Augmente les impayés |
