# Stratégie Pricing - Quickfund

> **Dernière mise à jour :** Décembre 2025

---

## Philosophie

| Élément | Réponse |
|---------|---------|
| **Philosophie** | Risk-based pricing (taux ajustés au profil) |
| **Position** | Aligné / légèrement supérieur (on prend plus de risques) |
| **Message** | Accessibilité + Transparence : "pas de frais cachés" |
| **Pricing power** | Moyen (profils mal servis = moins de concurrence) |
| **Stratégie** | Équilibre volume/marge |

**Principes :**
1. Transparence totale (TAEG, frais, pénalités)
2. Pas de frais cachés
3. Explication du taux
4. Pas d'usure (respect plafonds légaux)

---

## Taux d'intérêt

**Modèle :** Taux fixe, affiché en TAEG.
**Calcul :** Taux de base + prime de risque + marge.

### Variables d'influence

| Variable | Impact |
|----------|--------|
| Score crédit | Principal facteur |
| Montant | Micro = taux plus élevé |
| Durée | Court = TAEG plus élevé |
| Type produit | Pro souvent moins risqué |
| Historique | Client récurrent = réduction |
| Garanties | Leasing = taux avantageux |

### Fourchettes par produit

| Produit | TAEG indicatif |
|---------|----------------|
| Micro-crédit (20-500€) | 15-25% |
| Crédit conso (500-5 000€) | 8-18% |
| Prêt étudiant | 6-12% |
| Avance salaire | 10-20% |
| Crédit pro | 7-15% |
| Leasing | 5-12% |

*Fourchettes indicatives, à affiner Q1 2026.*
