## 1. Philosophie de pricing

**Questions à répondre :**

- [x] **Quelle est votre philosophie de pricing ?**
- [x] **Comment positionnez-vous vos prix vs la concurrence ?**
- [x] **Quel est le message pricing que vous voulez transmettre ?**
- [x] **Avez-vous un "pricing power" ?**
- [x] **Quelle est votre stratégie de capture de valeur ?**

### Réponses

| Élément | Réponse |
|---------|---------|
| **Philosophie** | **Risk-based pricing** : taux ajustés au profil de risque du client. Pas de taux unique, mais une fourchette selon le scoring. |
| **Position vs concurrence** | **Aligné / Légèrement supérieur** : pas le moins cher (on prend plus de risques en acceptant des profils atypiques), mais jamais abusif. |
| **Message pricing** | **Accessibilité + Transparence** : "On vous dit le coût réel, pas de frais cachés, et on vous explique pourquoi ce taux." |
| **Pricing power** | **Moyen** : capacité à maintenir les marges grâce au positionnement sur les profils mal servis (moins de concurrence directe). |
| **Stratégie capture valeur** | **Équilibre volume/marge** : maximiser le nombre de clients tout en maintenant une marge saine pour absorber les défauts. |

### Principes directeurs

1. **Transparence totale** : le client sait exactement ce qu'il paie (TAEG, frais de dossier, pénalités éventuelles)
2. **Pas de frais cachés** : tout est affiché upfront avant signature
3. **Explication du taux** : si le taux est élevé, on explique pourquoi (profil risqué, durée courte, etc.)
4. **Ajustement au risque** : plus le risque est faible, plus le taux l'est aussi
5. **Pas d'usure** : respect strict des plafonds légaux (taux d'usure français, réglementation estonienne)

## 2. Structure des taux d'intérêt

**Questions à répondre :**

- [x] **Quel est votre modèle de taux ?** (Fixe, variable, hybride)
- [x] **Comment calculez-vous le TAEG par produit ?**
- [x] **Quelles variables influencent le taux ?**
- [x] **Avez-vous des fourchettes de taux par produit ?**

### Modèle de taux

| Caractéristique | Réponse |
|-----------------|---------|
| **Type de taux** | Taux fixe (pas de variable sur la durée du prêt) |
| **Affichage** | TAEG (Taux Annuel Effectif Global) incluant tous les frais |
| **Calcul** | Taux de base + prime de risque selon scoring + marge |

### Variables influençant le taux

| Variable | Impact sur le taux |
|----------|-------------------|
| **Score de crédit** | Principal facteur : profil A → taux bas, profil C → taux élevé |
| **Montant emprunté** | Micro-montants = taux plus élevé (coût fixe à amortir) |
| **Durée** | Durées courtes = taux nominal plus bas mais TAEG potentiellement plus élevé |
| **Type de produit** | Crédit pro souvent moins risqué → taux plus bas |
| **Historique client** | Client récurrent avec bon historique → réduction possible |
| **Garanties** | Leasing avec garantie = taux plus avantageux |

### Fourchettes de taux par produit

| Produit | TAEG indicatif | Commentaire |
|---------|----------------|-------------|
| **Micro-crédit (20-500€)** | 15-25% | Taux élevé car montants faibles et durées courtes |
| **Crédit conso (500-5 000€)** | 8-18% | Fourchette standard, dépend du profil |
| **Prêt étudiant** | 6-12% | Taux préférentiels pour encourager l'éducation |
| **Avance sur salaire** | 10-20% | Court terme, remboursement garanti par salaire |
| **Crédit pro (1 000-10 000€)** | 7-15% | Profils souvent plus stables |
| **Leasing** | 5-12% | Garanti par le bien, donc risque réduit |
| **Rachat de crédits** | Variable | Dépend de la situation globale du client |

*Note : Ces fourchettes sont indicatives. Les taux exacts seront affinés en Q1 2026 lors de la structuration.*

### Types de promotions

| Promotion | Description | Fréquence |
|-----------|-------------|-----------|
| **Frais de dossier offerts** | 0€ de frais pour les nouveaux clients | Lancement, acquisition |
| **Taux promotionnel** | -1 à -2 points sur le TAEG standard | Périodes clés (rentrée, fêtes) |

| Niveau | Discount max autorisé | Qui peut l'accorder |
|--------|----------------------|---------------------|
| **Standard** | Frais de dossier offerts | Commercial |
| **Négociation** | -0.5 point TAEG | Manager |
| **Exception** | -1 point TAEG | Fondateur |

### Promotions à éviter

- ❌ Taux "0%" trompeurs (avec frais cachés)

- ❌ Promotions non rentables long terme
- ❌ Discounts agressifs qui dévaluent la marque

- Passage en contentieux = dernier recours

## 11. Objections et négociation

**Questions à répondre :**

- [x] **Quelles objections prix entendez-vous ?**
- [x] **Comment y répondez-vous ?**

### Objections fréquentes

| Objection | Fréquence | Réponse type |
|-----------|-----------|--------------|
| *"C'est trop cher / taux trop élevé"* | Fréquente | "Notre taux reflète notre capacité à vous accepter là où d'autres vous refusent. On prend un risque pour vous, et on vous accompagne avec le coaching inclus." |
| *"Ma banque propose moins cher"* | Moyenne | "Votre banque vous a accepté ? Si oui, foncez ! Si non, on est là justement pour ceux que les banques refusent." |
| *"Concurrent X est moins cher"* | Rare | "Comparez le TAEG total avec tous les frais. Vérifiez aussi s'ils vous acceptent et en combien de temps." |
| *"Je veux négocier le taux"* | Moyenne | "Le taux est calculé sur votre profil. Ce qu'on peut faire : frais de dossier offerts si vous signez aujourd'hui." |

### Battle card pricing

**Arguments Quickfund :**
1. Transparence totale (pas de frais cachés)
2. Coaching financier inclus (valeur ajoutée)
3. Acceptation de profils refusés ailleurs
4. Rapidité (<24h)
5. Possibilité de micro-montants

**Points de négociation possibles :**
- Frais de dossier
- Date de prélèvement
- Report 1ère échéance

**Points non négociables :**
- Taux en dessous du coût
- Suppression des pénalités de retard
- Conditions légales
